📰 Latest Angular News

­

Selectorless Components Proposal 👀

­

The Angular team recently shared an initial proposal for a selectorless component and directives in templates. The goal of the new approach is to reduce boilerplate, speed up compilation and shrink runtime size. You can read more about the idea behind the selectorless approach here .

­

Angular Won’t Force the .ng Suffix 🙅

­

After controversy over changing the .component suffix to .ng , the Angular team listened to community feedback and scrapped the plan. The updated style guide will remain neutral on identifier and file name suffixes. By default, the Angular CLI won’t generate suffixes, though the ng generate schematics, but it’ll still offer the option to specify them.

­

From Developer Preview to Stable ✅

­

In Angular 20, features such as effect , linkedSignal , toObservable , toPromise , toSignal , and afterNextRenderEffect are transitioning from Developer Preview to Stable. With these APIs, Angular signals offer a fully stable approach to building apps with most of the signal utilities.

­

Deprecation of Structural Directives 🧹

­

Structural directives like *ngIf , *ngSwitch , and *ngFor will be deprecated in Angular 20, aligning with a move toward a control flow-based approach as the recommended method for building Angular applications.

­

Zoneless Enters Developer Preview ✨

­

Angular’s Zoneless API is advancing from the experimental to the developer preview stage . The provideExperimentalZonelessChangeDetection API will be renamed to provideZonelessChangeDetection in Angular 20, slowly shaping the way for a future where Angular apps operate without a dependency on zone.js.

🌍 Community Corner

­

NgRx 19.1 🎉

­

This release brings three major updates: the withFeatureApi , which enables more flexible custom NgRx signal store features; an unprotected utility that streamlines signal store testing; and new entity updaters - prependEntity , prependEntities , upsertEntity , and upsertEntities . See the overview here .

­

Tanstack Router in Angular 🧭

­

Over the past month, Nart, Brandonroberts, and Birkskyum have been experimenting with the Tanstack Router , developing an adapter prototype for Angular. TanStack Router is a fully type-safe framework-agnostic routing library. It contains features such as support for data loading, caching, and declarative route definitions.

 

Recently tanstack tools are becoming increasingly available in the Angular ecosystem - especially Tanstack Query which is gaining more popularity each week.

­

Zod 4.0 Beta 🧪

­

Zod, the validation schema library, has released its new major version with plenty of community-requested features. The authors have significantly optimized runtime performance in the new version, reducing the bundle size by ~57% for zod/core and ~85% for the new lightweight @zod/mini version.

­

RsPack and Angular 🦀

­

If your application suffers from long build times and you can’t migrate to esbuild builder because of custom webpack plugin dependencies, consider switching your app builder to RsPack. Although support is still currently experimental, it is in a state that may be sufficient for your current needs. Read more in the recently published article by Nx on migrating from webpack to RsPack .

­

Upcoming Nx 21 Release 🔥

­

According to Victor Savkin , we can expect Nx version 21 in April. This release will upgrade many core aspects of Nx, including a new UI that allows running tasks in parallel and using keyboard shortcuts - all of that within one terminal window. The preview looks exciting.

📰 Latest Angular News

­

New RFC Related to Animations 🧩

­

The Angular team announced an RFC to deprecate @angular/animations in favor of native, CSS-powered animations for better performance and smaller bundles. They propose adding 2 new APIs, animate(dot)in and animate.out . The team also published a migration guide to css animations.

 

RFCs on Component Authoring Format, Signals Forms, and Selectorless Components haven’t been published yet, but some ideas were shared at the NG-Rome conference.

­

June Angular Q&A Session 💬

­

The Angular team answered community questions in the June Q&A . Topics included a discussion on file suffixes and effect function - the latter being one of the most commonly misunderstood aspects of Angular Signals. Angular provided guidance on when it should and shouldn’t be used.

­

Angular Documentation for AI Tooling 📄

­

Angular released official documentation files for AI tooling. New LLM prompts and AI IDE setup in official documentation includes custom prompts, IDE setup guides, and the new llms.txt standard.

🌍 Community Corner

­

Jest 30 Release 🧪

­

The new version of Jest has been officially released, bringing major upgrades like 37% faster performance, 77% lower memory usage, and improved ESM and TypeScript support. Performance gains may vary by project. New features include the expect.arrayOf matcher, the using keyword for automatic spy cleanup, and much more.

­

Vitest 3.2 Release ⚡

­

The new Vitest version brings improvements to browser module handling and TypeScript support. It also introduces a new annotations API for custom messages and attachments, and deprecates workspace config to simplify project setup.

­

Self Healing CI 🛠️

­

Nx introduced a new Nx Cloud feature: Self-Healing , an AI-powered solution that diagnoses and fixes errors in your CI pipeline. It works through a smooth review-and-approve model - automating fixing small test failures and other common issues.

­

Ng Baguette Conf 2025 Recordings 🎥

­

Recordings from the ng-baguette conference are now publicly available! Learn from experts as they share insights on data fetching, monorepos, SSR, and navigation in Angular. You can watch them here .
